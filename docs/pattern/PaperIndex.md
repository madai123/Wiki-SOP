# A. Index 模板  

当前索引文件为根目录 `INDEX-paper.md`。如使用 Dataview，优先维护 `wiki/papers/paper_id.md` 的 frontmatter，由索引自动展示。

## Paper Index 表格格式  
  
### 字段说明  
  
- `Paper_id`：引用到具体 paper summary 的反链，必须使用 `[[paper_id]]` 格式  
- `Title`：论文标题 
- `Year`：发表年份  
- `Source ID`：外部来源编号，例如 arXiv ID、DOI、Semantic Scholar ID  
- `status`：论文处理状态，包含`metadata finished / summary finished / have discussed`
- `check`：人工审核，包含 `checked / uncheck`  
- `Notes`：简明记录
